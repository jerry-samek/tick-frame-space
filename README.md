# Tick-Frame Space: Dimensional Physics Experiments

Welcome to the Tick-Frame Space project. This repository contains documentation, theory, and experiment scripts for investigating dimensional phase transitions, saturation boundaries, and stability-probability balance in discrete physics models.

---

## Usage

- Treat the system as a **3D modeling playground** for expanding voxel grids.
- Experiment with entity persistence, renewal cycles, and emergent structures.
- Extend rulesets to explore different propagation laws (uniform dilution, causal weighting, renewal efficiency).
