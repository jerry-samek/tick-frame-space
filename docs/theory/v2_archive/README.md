# V2 Theory Archive

**Version 2** was written January-February 2026 and consolidated the first ~76 raw documents
into 13 chapters plus supporting documents.

## V2 Core Assumption (Superseded)

> Time is the primary substrate. Entities are temporal processes on a tick-stream.
> Space is emergent from temporal gradients. Geometry is Euclidean at the Planck scale.
> Connectors are edges on a discrete lattice.

This assumption was superseded by the graph-first framework established in RAW 108-113
(February-March 2026). See `docs/theory/V3_README.md` for the current version.

## What Changed

| Concept | V2 | V3 |
|---------|----|----|
| Substrate | Discrete Euclidean lattice | Raw graph, no geometry |
| Space | Emergent from temporal gradients | Emergent from observer latency matrix |
| Time | Primary substrate (tick-stream) | Branch depth - observer property |
| Connectors | Edges on fixed grid | Deposit chains |
| Dimension | Substrate property | Observer property |

## What Survives

Many V2 results survive the substrate change because they test mechanisms, not substrates:

- **rho=2.0 signature** (Exp #50) - time is special, substrate-independent
- **Rotation asymmetry 933x** (Exp #44) - kinematic constraint, substrate-independent
- **O(n) bucketing** (Exp #44_05) - discrete-time property, substrate-independent
- **Observer model** (Ch4) - entirely conceptual, no substrate dependency
- **Free will framework** (Ch5) - philosophical, no substrate dependency

Results that need re-validation on graph substrate:
- 3D optimality (Exp #15) - tested on regular grids only
- Time dilation r=0.999 (Exp #51) - continuous field substrate
- Collision physics (Exp #55) - lattice substrate
