# Review: Transitional Documents

These documents were written during or after the paradigm shift from geometric to
graph-first substrate (roughly RAW 024–069, plus RAW 996). Their **conclusions are
probably valid** but their substrate language may need updating.

## Why These Are Here

Each document in this directory meets one or more of these criteria:

- Has conclusions that likely survive the graph-first substrate change but uses
  geometric language (lattice, Planck units, Euclidean)
- Is an experimental result document (results valid even if substrate model changed)
- References both geometric and graph concepts (transitional era)
- Was uncertain during triage — safer to under-archive than over-archive

## What Needs Doing

Each document should be reviewed individually to determine:

1. **Does the core conclusion depend on geometric substrate?** If yes → `archive/`
2. **Can the conclusion be restated in graph-first terms?** If yes → update and return to `raw/`
3. **Is it an experimental result?** Results are substrate-specific but mechanism may transfer
