# Archive: Geometric-Era Documents

These documents were written during the **geometric substrate era** (roughly RAW 001–027,
plus the `.tex` duplicate of RAW 039). They derive physics from a discrete Euclidean lattice
with Planck-scale voxels. This substrate model has been superseded by the graph-first
framework established in RAW 108–113 (February–March 2026).

## How to Read These Documents

The **substrate descriptions** in these documents are outdated. Connectors are not geometric
edges — they are deposit chains (RAW 111). Distance is not Planck units — it is observer-
reconstructed from causal latency (RAW 040, RAW 110). Geometry is not a substrate
primitive — it is an emergent observer property.

However, some **conclusions** from these documents survive the substrate change:
