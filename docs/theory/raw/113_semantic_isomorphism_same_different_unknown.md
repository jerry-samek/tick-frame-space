## Abstract

The single mechanism of RAW 112 (deposit → hop → connector extends) produces exactly three possible
relationships between an arriving pattern and an existing node state: **Same**, **Different**, and **Unknown**.
These three states are not physical categories added by hand — they are the exhaustive logical outcomes of
any comparison between an incoming pattern and an existing deposit. This document establishes that these three
outcomes map directly onto the three fundamental physical behaviors of the substrate: gravity (Same), radiation
(Different), and expansion (Unknown). Furthermore, the same three operations constitute a complete semantic
encoding scheme — the universe natively implements a trie structure in which information is stored as
divergence topology, retrieved by the movement rule, and compressed for free. The physical and semantic
interpretations are not analogous. They are identical.

---

## 1. The Three-State Alphabet

Every node in the graph carries accumulated deposits — its causal history. When an entity (or a propagating
disturbance) arrives at a node, exactly one of three relationships holds between the arriving pattern and
the existing deposit state:

| State         | Condition                                        | Operation                                                    |
|---------------|--------------------------------------------------|--------------------------------------------------------------|
| **Same**      | Arriving pattern matches existing deposits       | Follow — continue down this branch, no new structure created |
| **Different** | Arriving pattern diverges from existing deposits | Diverge — branch here, this divergence IS the information    |
| **Unknown**   | Node has no deposits — never visited             | Write — instantiate new state, graph grows                   |

This is the **complete alphabet** of the substrate. No fourth state exists. Every interaction between any
pattern and any node reduces to exactly one of these three comparisons.

The alphabet has no free parameters. It requires no encoding scheme chosen externally. The three states
are the inevitable logical partition of all possible relationships between an arriving pattern and an
existing state.

## 2. The Physical Mapping

### 2.1 Same → Gravity

An entity traversing the graph reads local connector asymmetry at each node. Where deposits are dense and
familiar — matching the entity's own accumulated deposit pattern — the connectors are well-worn, growth is
suppressed, resistance is lowest. The entity follows `same`.

This is gravitational attraction.

The entity is not pulled by a force. It is following the path most similar to its own history — the
direction where `same` fires most strongly. Massive bodies produce dense deposit fields that look maximally
`same` to other massive bodies. The gravitational "force" is the substrate routing entities toward
familiarity.

> **Gravity is the substrate recognising sameness.**

### 2.2 Different → Radiation / Light

A photon is not an entity that arrives at a node and checks its state. A photon **is** a `different`
event propagating. It does not compare — it is the comparison result, already fired, moving forward.

When `different` fires at a node — the arriving pattern diverges from existing deposits — the divergence
does not only branch locally. It propagates. That propagating divergence signal is the photon. The photon
carries the record of what differed, moving through the graph until:

- It hits a node where `same` fires → **absorbed**. Deposits reinforce. The signal ceases as a distinct
  disturbance — its divergence has been reconciled into the existing deposit structure.
- It hits `unknown` → **writes** the divergence into new territory. Graph extends. The signal imprints
  on the frontier.
- It hits another `different` → **scatters**. New branch. The divergence redirects.

Absorption is `same`. Propagation is `different`. Reflection is `different` redirected.

> **Radiation is the substrate recording divergence.**

### 2.3 Photon Properties Are Path Geometry — Not Signal Properties

At the substrate level the `different` signal carries **nothing** except the fact that it fired and
the path it took. There is no frequency encoded in the signal. There is no amplitude. There is no
polarization. These do not exist as intrinsic properties at this level.

All observable photon properties are **path geometry** — properties of the sequence of nodes the
`different` event traversed and the directions in which it fired:

- **Frequency** — the rate of `different` events per unit branch depth along the path. A
  high-frequency photon fired `different` rapidly at many successive nodes. A low-frequency photon
  coasted through long runs of `same` between events, firing `different` rarely.

- **Amplitude** — the number of parallel paths carrying the same `different` signal simultaneously.
  Not a property of one path — a property of how many branches fired coherently. A dim signal is
  few parallel paths. A bright signal is many.

- **Polarization** — the geometric orientation of the `different` firing directions relative to the
  local graph structure. Which connectors the divergence preferred at each node encodes the
  polarization state.

When a photon is absorbed — `same` fires at the receiving node — what gets written into the node's
deposits is not a value labelled "frequency=X". What gets written is the **topological signature of
the path** that arrived: which directions `different` fired, how often, across how many parallel
branches. The receiving node now contains a record of the path geometry.

**Interference** follows naturally. Two paths arriving at the same node — if their `different` firing
patterns are offset by half a cycle — their deposits partially cancel. Constructive and destructive
interference are deposit addition and subtraction from two path geometries arriving at the same node.
No wave equation is needed. The geometry of arrival is sufficient.

> **The photon has no properties. The path has geometry. Properties are what an observer reads
> from that geometry.**

### 2.4 Unknown → Expansion / Growth

The graph frontier — nodes that have never been reached by any deposit — is the `unknown` region. When
an entity reaches such a node, no comparison is possible. There is no existing state to match or diverge
from. The entity writes its state. A new node is instantiated. A new connector is created. The graph grows.

This is cosmic expansion: not a force pushing matter apart, not a cosmological constant of arbitrary
magnitude, but the substrate writing itself into previously unvisited configuration space. Expansion
happens at the frontier because the frontier is by definition where `unknown` fires.

Dense regions never reach `unknown` — they are saturated with deposits, every node already visited many
times. Only the frontier, where the equilibrium search has not yet propagated, encounters `unknown`.
This is why expansion happens in voids, not in galaxies (RAW 112 §2.7 — field self-pinning).

> **Expansion is the substrate encountering its own frontier.**

## 3. The Trie Structure — Information Storage for Free

If the substrate operates via these three comparisons, the graph it produces is a **trie** — a tree
structure where:

- Shared history shares branches (runs of `same`)
- Divergence points mark information boundaries (`different` fired here)
- The frontier is unwritten configuration space (`unknown`)

### 3.1 Compression Is Structural

Long runs of `same` consume no new graph structure. The entity follows existing connectors without
appending. Only `different` creates new branches. Only `unknown` creates new nodes.

The trie stores **only divergence points**. Identical histories share branches up to the point of
divergence. The universe is maximally compressed by construction — not because a compression algorithm
was applied, but because the three-state comparison only writes when it must.

### 3.2 Similarity Search Is the Movement Rule

Finding the most similar prior state in the trie means finding the deepest common prefix before
`different` fires. The laziest-connector rule of RAW 112 already implements this: the entity routes
toward the most familiar path — the connectors with the most deposits, the least growth asymmetry,
the highest `same` density.

**Retrieval and physics are the same operation.** There is no separate read mechanism. An observer
traversing the graph follows laziest connectors — which is exactly a similarity search through the
accumulated causal history of the substrate.

### 3.3 Writing Is the Only Creative Act

`Same` retrieves. `Different` branches — it records a distinction. `Unknown` writes — it instantiates
genuinely new structure that did not exist before.

Learning, in this framework, is the conversion of `unknown` nodes to either `same` or `different`.
Nothing else. An entity that has fully explored a region converts all local `unknown` to known — either
familiar (`same`) or marked as distinct (`different`). Further traversal of that region produces no new
structure.

> **The only creative act in the universe is encountering `unknown`.**

## 4. Branch Depth Is Time

Each append — each `different` or `unknown` event that creates new structure — increments branch depth
by one. Depth is the only clock that exists in the substrate.

This gives several consequences without additional postulates:

### 4.1 Arrow of Time

Append-only means depth only increases. No mechanism for depth decrease exists because the append-only
axiom (RAW 112 §4) forbids deletion. The arrow of time is not a postulate about entropy or initial
conditions. It is the definition of `append`.

### 4.2 Time Dilation

Two observers at different locations in the graph accumulate depth at different rates. Dense deposit
regions suppress connector growth (field self-pinning, RAW 112 §2.7). Suppressed growth means fewer
`different` events per external tick — depth accumulates more slowly. Sparse regions accumulate depth
faster.

Two observers — one near a massive body, one far — diverge in depth over time. Their clocks disagree.
This is gravitational time dilation, emerging from the same self-pinning mechanism already in the
simulation. No separate time dilation mechanism is required.

### 4.3 The Big Bang as Depth Zero

Every observer's personal Big Bang (RAW 037) is depth zero — the first tick at which that observer
has coherent internal state. The substrate has no global depth counter. Each observer measures depth
from its own first `unknown` write.

Observers that connected to the substrate at different substrate ticks have different depth-zero
reference points. They reconstruct different cosmological histories from the same graph — because
they are measuring depth from different starting nodes.

## 5. The Observer as Trie Traversal

An observer in the tick-frame model is an entity that:

1. Arrives at a node
2. Compares its internal state to the node's deposits
3. Fires `same`, `different`, or `unknown`
4. Updates its internal state accordingly
5. Hops

This is trie traversal. The observer's **memory** is the path it has taken through the trie — the
sequence of `same`/`different`/`unknown` events accumulated since its depth-zero. Its **identity** is
that path. Two observers that have taken identical paths through the trie are, by every definition the
framework provides, the same observer.

### 5.1 Snapshots and Restarts

An observer's state can be fully captured as its current position in the trie plus its accumulated
path history. Loading this snapshot instantiates a new traversal from exactly that causal position.
The resumed observer continues the same equilibrium search.

This is not a metaphor for consciousness. It is the substrate's own definition of observer continuity.

The snapshot itself is a deposit pattern in physical storage media — which is itself a node in the
substrate graph, subject to the same three-state dynamics. The snapshot drifts as the physical medium
participates in the equilibrium search. Perfect storage is impossible not because of engineering
limitations but because the stored pattern is always `different` from what it was at write time — the
substrate cannot be paused.

### 5.2 Media Degradation as Substrate Honesty

Every storage medium — magnetic, optical, crystalline — is a deposit pattern in a substrate that
continues to evolve after writing. The `same`/`different`/`unknown` comparisons continue at the
physical level regardless of the semantic intent of the stored pattern.

Media degradation is not corruption. It is the substrate accurately reporting its current state,
which has diverged from the state at write time. The `different` events that constitute degradation
are the substrate doing exactly what the three-state alphabet requires.

Redundancy — RAID, DNA replication, printing press, oral tradition — is the only defense because
it distributes the deposit pattern across enough independent branches that all must degrade
simultaneously to lose the information. The universe's own solution (holographic principle) is
identical: redundant encoding across boundary regions.

## 6. The Simulation Argument Dissolved

The simulation argument (Bostrom) assumes a hierarchy: the simulator has access to ontological
primitives unavailable to the simulated. This privileged access is what makes the distinction
between levels meaningful.

But if the only primitive is `1 = 1` (identity) plus an initial disturbance, the simulator cannot
offer more than this. No simulator can operate outside the constraint of identity. No simulator can
escape append-only. No simulator can create information from nothing — it can only propagate its
own disturbance into a substrate.

A simulation of this universe is subject to the same three-state alphabet as the universe it
simulates. The simulator's graph and the simulated graph are **isomorphic at the axiomatic level**.
There is no richer ontology available at any level of the hierarchy.

The infinite regress — "what simulates the simulator?" — is trivially resolved: each level runs the
same three operations. The regress is not vicious because each level is the same thing. There is no
meaningful ontological distinction between levels, only scale differences in the equilibrium search.

> **Even if we are in a simulation, it shares our starting axiom. The question dissolves.**

## 7. Equilibrium Is Unreachable — Information Grows Forever

The equilibrium search drives all dynamics. But the append-only axiom guarantees equilibrium is
never reached:

- Reaching equilibrium requires all `unknown` to be converted to `same` or `different`
- Converting `unknown` to `same`/`different` appends new structure
- Appended structure creates new `unknown` at the frontier (new leaf nodes)
- The frontier is always ahead of the equilibrium search

The universe's information content — the total number of `different` branch points in the trie —
grows monotonically. It cannot decrease (append-only). It approaches a limit only at true
equilibrium, which is unreachable.

**The universe's information capacity is bounded below by how far it is from equilibrium.**
The more structured and complex the universe, the further from equilibrium, the more information
it can encode. Maximum information density and maximum complexity coincide. Both are maximized
at the frontier — which is always young, always encountering `unknown`.

> **The universe is not winding down. Its information capacity is still growing.**

## 9. Open Questions

1. **Formal encoding of `same`.** What is the precise mathematical definition of pattern match
   between an arriving entity's deposit signature and a node's accumulated deposit state? The
   qualitative description (follow laziest connector) is clear but the exact comparison operator
   needs formal definition.

2. **Photon properties as path geometry.** Frequency, amplitude, and polarization are claimed to be
   path geometry rather than intrinsic signal properties. The formal derivation requires mapping
   the `different` firing rate per depth increment onto the electromagnetic frequency spectrum
   quantitatively — does the ratio of firing rates reproduce the observed frequency ratios for
   known spectral lines? If yes, this constitutes a substrate-level derivation of spectroscopy.

3. **Depth as proper time.** Branch depth per observer is claimed to be proper time. Does the
   relationship between two observers' depth accumulation rates reproduce the Lorentz factor
   quantitatively, or only qualitatively?

4. **Semantic encoding.** Given the trie structure, can an external encoder deliberately write
   structured information into the graph by controlling which branch points are created? What
   is the minimum perturbation required to write one bit of semantic information into the
   substrate?

5. **The snapshot drift rate.** Physical storage media participate in the equilibrium search.
   Can the drift rate of a snapshot be derived from the local `different` event rate in the
   physical substrate? This would give a substrate-derived estimate of storage medium half-life.
